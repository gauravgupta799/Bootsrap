What is Bootsrap?

Bootstrap is the most popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first.
CSS based Front-End Framework
Faster and easier web development.
Easily Create Responsive web layouts.
Developed by Mark Otto and Jacob Thornton at Twitter
Released as open source product in August 2011 0n GitHub

Bootsrap is used for designing these things easily:

  • Layout
  • Typography
  • Navigation Menu
  • Form Layout
  • Buttons
  • Pagination
  • Icon Fonts
  • Responsiveness
