Bootstrap Table

  • .table-striped -> Zebra stripes to a table
  • .table-bordered -> Adds boder on all sides of the table and cells.
  • .table-hover -> Adds a hover effect with grey background on table rows.
  • .table-condensed -> More compact by cutting cell padding in half.
  • .active -> Hover to table row or cell.
  • .success -> Successful or positive action
  • .info -> Neutral informative change or action.
  • .warning -> Might need attention
  • .danger -> dangerous or potentially negative action
  • table-responsive
